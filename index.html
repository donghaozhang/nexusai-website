Frequently Asked Questions

Is QCut really free?
Yes! QCut is 100% free and open source under the MIT license. No subscriptions, no watermarks, no hidden fees. You only pay for AI API usage if you use features like Sora or FLUX.
How does the AI pricing work?
You bring your own API keys (FAL AI, ElevenLabs, etc.) and pay only for what you use. No markup ‚Äî you pay the same rates as direct API access. A typical video generation costs $0.40-2.00.
Is my data private?
Yes. All video editing happens locally on your device using FFmpeg WebAssembly. Your videos never leave your computer. AI features communicate directly with API providers ‚Äî QCut never sees your content.
What platforms are supported?
Currently Windows. macOS and Linux builds are planned. QCut is built with Electron, so cross-platform support is on the roadmap.
Can I contribute to QCut?
Absolutely! QCut is open source. Check out our GitHub repository to report bugs, suggest features, or submit pull requests.
