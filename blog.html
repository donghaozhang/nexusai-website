Quick Start

1. Download & Install

# Download from GitHub Releases

https://github.com/donghaozhang/qcut/releases

2. Or Build from Source

# Clone the repository

git clone https://github.com/donghaozhang/qcut.git

cd qcut

# Install dependencies

bun install

# Build and run

cd qcut/apps/web && bun run build

cd ../.. && bun run electron

3. Start Editing!

Import your media, drag to timeline, and start creating.

AI Features Setup

To use AI content generation features, you'll need to configure API keys for the services you want to use.

Required API Keys

FAL AI Most Models

Required for image/video generation (Veo, Sora, Kling, FLUX, etc.)

Get API Key → fal.ai/dashboard
ElevenLabs TTS

Required for text-to-speech and voice features

Get API Key → elevenlabs.io/app/settings
Freesound Optional

Required for sound effects library search

Get API Key → freesound.org/apiv2/apply

CLI Reference

The AI content pipeline can also be used standalone via CLI:

# Install the AI pipeline

pip install video-ai-studio

# List available models

ai-content-pipeline list-models

# Generate an image

ai-content-pipeline generate-image --text "epic sunset" --model flux_dev

# Create a video

ai-content-pipeline create-video --text "dragon flying over mountains"

# Run custom pipeline

ai-content-pipeline run-chain --config pipeline.yaml
