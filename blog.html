Quick Start

1. Download & Install

# Download from GitHub Releases
https://github.com/donghaozhang/qcut/releases

2. Or Build from Source

# Clone repository
git clone https://github.com/donghaozhang/qcut.git
cd qcut

# Install dependencies
bun install

# Build and run
cd qcut/apps/web && bun run build
cd ../.. && bun run electron

CLI Reference

The AI pipeline can be used standalone:

# Install
pip install video-ai-studio

# List models
ai-content-pipeline list-models

# Generate image
ai-content-pipeline generate-image --text "sunset" --model flux_dev

# Create video
ai-content-pipeline create-video --text "dragon flying"
