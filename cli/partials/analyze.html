Analysis Commands

Analyze, transcribe, query, and translate videos with AI.

analyze video

Analysis

Analyze a video with AI vision.

qcut analyze video -i video.mp4 --analysis-type summary
FlagShortTypeDescription
--input-istringVideo file or URL required
--model-mstringVision model (default: gemini_qa)
--analysis-typestringtimeline, summary, description, transcript
--promptstringCustom prompt (overrides analysis-type)
--output-format-fstringmd, json, both

analyze transcribe

Analysis

Transcribe audio to text with optional SRT subtitle output.

qcut analyze transcribe -i audio.mp3 --srt --language en
FlagShortTypeDescription
--input-istringAudio file or URL required
--model-mstringSTT model (default: scribe_v2)
--languagestringLanguage code (e.g. en, fr)
--srtbooleanGenerate .srt subtitle file
--srt-max-wordsintegerMax words per SRT block
--no-diarizebooleanDisable speaker diarization
--keytermsstring[]Domain keywords (repeatable)

analyze query

Analysis

Query video segments for keep/cut analysis.

qcut analyze query -i video.mp4 -t "Find all action scenes"
FlagShortTypeDescription
--input-istringVideo file or URL required
--promptstringQuery prompt
--text-tstringAlias for prompt
--model-mstringVision model (default: gemini_qa)

analyze translate

Analysis

Translate video or audio into another language using HeyGen Translate. Audio input (.mp3, .wav, etc.) is automatically supported.

qcut analyze translate -i video.mp4 -l Spanish
FlagShortTypeDescription
--input-istringVideo or audio file/URL required
--language-lstringTarget language required
--output-ostringOutput directory
--audio-onlybooleanTranslate audio only (keep original video)
--output-audiobooleanOutput translated audio file (auto for audio input)
--speakersnumberNumber of speakers
qcut analyze translate -i podcast.mp3 -l Chinese
