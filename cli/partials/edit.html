Editing Commands

Autoclip, upscale, motion transfer, and compositing.

edit autoclip

Editing

Extract highlight clips from a video using subtitle-based LLM analysis. 4-step pipeline: outline → timeline → scoring → ffmpeg cutting.

qcut edit autoclip -i video.mp4 -s subs.srt -o /tmp/clips --min-score 0.8
FlagShortTypeDescription
--input-istringInput video file required
--srt-file-sstringSRT/VTT subtitle file (auto-detects if omitted)
--output-ostringOutput directory for clips
--min-scorenumberMinimum score threshold 0-1 (default: 0.7)
--stepnumberRun only a specific step (1-4)
--dry-runbooleanRun analysis only, skip cutting

edit upscale

Editing

Upscale an image to higher resolution.

qcut edit upscale -i photo.png --target 2160p
FlagShortTypeDescription
--input-istringImage path or URL
--model-mstringUpscaling model (default: topaz)
--targetstring720p(1x), 1080p(2x), 1440p(3x), 2160p(4x)
--output-format-fstringOutput format (default: png)

edit motion

Editing

Transfer motion from a reference video onto an image.

qcut edit motion --image-url photo.png --video-url dance.mp4
FlagShortTypeDescription
--image-urlstringSource image required
--video-urlstringReference motion video required
--model-mstringModel key (default: kling_motion_control)
--no-soundbooleanStrip audio
--text-tstringMotion prompt

gen music

Generation

Generate full music tracks with vocals and lyrics using MiniMax Music v2.6. Supports instrumental mode, structure tags, and audio quality settings.

# With lyrics (supports [verse], [chorus], [bridge], [outro] tags)
qcut gen music -t "Indie folk, acoustic guitar, melancholic" \
  --lyrics "[verse] Walking through the rain alone tonight [chorus] But I keep on walking"
# Chinese traditional style with lyrics
qcut gen music -t "Chinese guzheng, bamboo flute, elegant female vocal, 80 BPM" \
  --lyrics "[verse] 月落乌啼霜满天 [chorus] 千里江山如画卷"
# Instrumental (no vocals)
qcut gen music -t "Cinematic orchestral, epic, Hans Zimmer style" --instrumental
FlagShortTypeDescription
--text-tstringMusic style/mood description (10-300 chars) required
--lyricsstringSong lyrics with structure tags: [verse], [chorus], [bridge], [outro]
--instrumentalbooleanGenerate vocal-free instrumental music
--model-mstringMusic model (default: minimax_music_v2_6)
--audio-formatstringOutput format: mp3, wav, pcm (default: mp3)
--sample-ratenumberSample rate: 16000, 24000, 32000, 44100
--bitratenumberBitrate: 32000, 64000, 128000, 256000

gen tts

Generation

Generate speech from text using Chatterbox, ElevenLabs, or Qwen3.

qcut gen tts -t "Hello world" --provider elevenlabs
FlagShortTypeDescription
--text-tstringText to speak required
--model-mstringTTS model (default: chatterbox_tts)
--providerstringProvider shorthand: chatterbox, elevenlabs, qwen
--voicestringVoice preset (ElevenLabs/Qwen3)
--audio-urlstringVoice reference audio URL (for cloning)
--language-codestringLanguage code (ElevenLabs)
